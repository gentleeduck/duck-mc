# Introduction

Welcome to the **duck-ui** docs. This page exercises every capability the
dmc compiler is expected to provide for an opinionated component-library
documentation site.

## Why duck-ui?

duck-ui is a compound-component pattern library: the host renders the
`<Tabs>` shell, and the parts (`<TabsList>`, `<TabsTrigger>`,
`<TabsContent>`) coordinate via shared context. Each part is composable,
keyboard-navigable, and accessible by default.

## Getting started

<Callout title="Heads up" tone="info">
Install via your package manager of choice. The CLI scaffolds your
component into the project under `components/ui/`.
</Callout>
