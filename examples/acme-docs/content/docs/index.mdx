## What is acme/ui?

acme/ui is a UI component library that actually works. It's part of the
[acme](https://github.com/acme) ecosystem — a collection of open-source tools built by developers
who are tired of fighting their frameworks instead of building great products.

We're here to solve real problems: components that don't break when you look at them wrong,
documentation that doesn't assume you have a PhD in computer science, and maintainers who
actually respond to issues.

## Why acme/ui

Most UI libraries fall into two camps: gorgeous but impossible to customize, or flexible but uglier
than a frontend developer's first CSS file. acme/ui refuses to pick a side.

**What you actually get:**

- **Themes that bend to your will** — light mode, dark mode, custom modes
- **Components built right** — Radix-grade a11y, sane keyboard navigation
- **Documentation that respects you** — copy-paste-ready snippets

```tsx
import { Button } from 'acme/ui'

export default function Demo() {
  return <Button>Click me</Button>
}
```
