# Kitchen Sink

Every native dmc transformer firing on one page. Compare against the velite render of the same `.mdx` for parity.

## Inline marks

Plain, **bold**, _italic_, ~~strikethrough~~, `inline code`, ***bold italic***, **_nested mixed_**.

Bold around a link: **[github](https://github.com) link**.

Italic around a link: _[mdn](https://developer.mozilla.org) link_.

Inline code with backticks: `let x = 1;` and `git diff --cached`.

Escaped chars: \*literal-asterisks\*, \_underscores\_, \`backtick\`, \\ backslash, \$ dollar.

## GFM

Tables, strikethrough, autolinks, task lists.

| feature | sidecar plugin | native transformer |
| ------- | -------------- | ------------------ |
| code highlight | `rehype-pretty-code` | `pretty-code` (syntect) |
| math | `remark-math` + `rehype-katex` | `math` (katex via quick-js) |
| emoji | `remark-emoji` | `emoji` (unicode lookup) |
| slug + autolink | `rehype-slug` + `rehype-autolink-headings` | `autolink-headings` |

Aligned table:

| left   | center  | right |
| :----- | :-----: | ----: |
| a      | b       | c     |
| **bold** | _italic_ | `code` |
| 1      | 2       | 3     |

## Code

Rust with title + line marks:

```rust title="lib.rs" {3,5}
fn main() {
    let answer = 42;
    println!("the answer is {answer}");
    let v = vec![1, 2, 3];
    println!("{:?}", v);
}
```

TypeScript:

```ts title="render.tsx"
import { type ReactNode } from "react";

export function Greeting({ name }: { name: string }): ReactNode {
  return <h1>Hello, {name}!</h1>;
}
```

JSON without title:

## Math

Inline: $E = mc^2$, and $\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$, and $\binom{n}{k} = \frac{n!}{k!(n-k)!}$.

Block:

$$
\int_{-\infty}^{\infty} e^{-x^2}\,dx = \sqrt{\pi}
$$

Aligned:

$$
\begin{aligned}
f(x) &= ax^2 + bx + c \\
f'(x) &= 2ax + b \\
f''(x) &= 2a
\end{aligned}
$$

Matrix:

$$
A = \begin{pmatrix}
a & b \\
c & d
\end{pmatrix}
$$

Cases:

$$
|x| = \begin{cases}
x & \text{if } x \ge 0 \\
-x & \text{if } x < 0
\end{cases}
$$
